\section{Introduction}

Welcome to Dalis! This is the Unix setup i have built over the years with
maximum efficiency in mind!\\
This manual will cover all bindings and configuration you can do!\\
Later i will explain how my personal scripts work and how the file system is organized.

\section{Before Starting}

Before you jump to this insane keyboard-centric workfow, i feel you must have some skills
like being familiar with the terminal, vim motions (most of my bindings are based on vim motions)
and the ability to change or add features to this build.\\
If you are not willing to take the time to understand these tools and prioritize
ease-of-use over simplicity then this might not be for you.\\
The objective of this is for other people to build their own system with a simple
starting point.

\section{Starting out}

If you are reading this you \textit{probably} installed the system already but
in the case you still haven't the next section is for you, otherwise just skip it.

\subsection{Running the wizard}

After installing one of the supported operating systems you can run its install wizard!
It is located in repo/install/(distro)-wizard.\\

\begin{itemize}
    \item You have the multilib repository enabled in pacman.conf
    \item Your user has can do commands as root using the "sudo" command
\end{itemize}

\section{Post Install}

After installing the script you may want to configure \$HOME/.dalis/config so you can
configure monitors, backups and other variables used in scripts.

You won't need to do this if you don't want any backups or all you have is a single 60hz monitor

\section{Bindings}
Here are all the bindings you need to know:

\subsection{Script Bindings}

\begin{itemize}
    \item Meta + Return = Open St Terminal
    \item Meta + n      = Lf file manager  
    \item Meta + N      = Lf file manager in your notes directory
    \item Meta + p      = Execute a binary avaliable from \$PATH
    \item Meta + P      = System config script
    \item Meta + e      = Emoji Selector
    \item Meta + E      = System Maintenance script
    \item Meta + w      = Open Web Browser
    \item Meta + W      = Open secondary Web Browser
    \item Meta + W      = Open secondary Web Browser
\end{itemize}

\subsection{Window Management Bindings}
\begin{itemize}
    \item Meta + t      = Switch to tile layout
    \item Meta + T      = Switch to bstack layout
    \item Meta + m      = Switch to monocle layout
    \item Meta + M      = Switch to centered master layout
    \item Meta + h      = Move window separator to the left
    \item Meta + l      = Move window separator to the right
    \item Meta + k      = Select next window
    \item Meta + j      = Select previous window
    \item Meta + K      = Move window to next space
    \item Meta + i      = Increase window's size on the stack
    \item Meta + d      = Decrease window's size on the stack
    \item Meta + F11    = Make a window Full Screen
    \item Meta + Shift + F11    = Toggle FakeFullScreen for the selected window
\end{itemize}
